# 培训学院页面布局说明

## 整体结构（自上而下）

```
┌─────────────────────────────────────────────────────────────────┐
│  site-header（全站通用）                                          │
│  [链域智汇] [首页] [专家智库] [项目成果] [培训学院] [登录/注册]          │
├─────────────────────────────────────────────────────────────────┤
│  main #app（培训学院内容由 app.js 渲染）                             │
│  ┌───────────────────────────────────────────────────────────┐  │
│  │ 1. 顶部横幅 academy-hero（深蓝主色）                         │  │
│  │ 2. 内容区 container academy-content                          │  │
│  │    ├ 热门直播                                               │  │
│  │    ├ 知识专区                                               │  │
│  │    ├ 精品课程                                               │  │
│  │    └ 行业报告                                               │  │
│  └───────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────────┘
```

---

## 1. 顶部横幅（academy-hero）

- **布局**：横向 flex，左右分栏，容器 `container academy-hero-inner`
- **左侧** `academy-hero-promo`（弹性占满）  
  - 品牌 + 徽标：链域智汇 · 供应链培训学院  
  - 标签：体系化课程、新人专享、专家辅导  
  - 文案：以专业方法构建韧性供应链  
  - 占位：二维码区域
- **右侧** `academy-hero-checkin`（固定宽度约 224px）  
  - 标题：学习签到  
  - 文案：连续签到 2 天  
  - 按钮：立即签到

---

## 2. 热门直播（academy-section）

- **标题**：热门直播  
- **副标题**：精选直播预告与精彩回顾  
- **布局**：`academy-live-grid`，横向 flex，两列  
  - **左** `academy-live-main`（主卡片，弹性）  
    - 副标题、标题、时间、图标区、按钮「马上观看」  
  - **右** `academy-live-sidebar`（侧栏，约 288px）  
    - Tab：直播预告 | 精彩回顾  
    - 列表：圆点 + 标题 + 标签（预告/回顾）

---

## 3. 知识专区（academy-section）

- **标题**：知识专区  
- **副标题**：行业知识与实操指南  
- **布局**：`academy-knowledge-grid`，2 列网格  
  - 每项 `academy-knowledge-card`：左侧图标（分类首字）+ 右侧正文（分类、标题、浏览量·日期）  
- **底部**：居中的「查看更多」按钮

---

## 4. 精品课程（academy-section）

- **标题**：精品课程  
- **副标题**：体系化课程，助力能力提升  
- **分类 Tab**：`academy-category-tabs` 横向排列  
  - 全部课程、新人、入门、计划、仓配、采购、数字化、风险  
- **布局**：`academy-course-grid`，3 列网格  
  - 每张卡片：封面图（含热销/新课/免费标签）+ 标题、讲师·节数、价格（原价划价 + 现价/免费）  
- **底部**：居中的「查看更多」按钮

---

## 5. 行业报告（academy-section）

- **标题**：行业报告  
- **副标题**：数据洞察与行业白皮书  
- **分类 Tab**：全部、运营、计划、仓配、数字化、数据报告  
- **布局**：`academy-report-grid`，3 列网格  
  - 每张卡片：顶部 PDF 色块（可带背景色）+ 标题、下载量·更新日期  
- **底部**：居中的「查看更多」按钮

---

## 响应式约定

- **≤720px**：  
  - 知识专区、精品课程、行业报告网格改为 1 列  
  - 直播侧栏宽度取消限制（占满）  

---

## 文件与类名对应

| 区域           | 主要 class                     | 说明           |
|----------------|--------------------------------|----------------|
| 整页容器       | `.academy-page`               | 培训学院整页   |

| 热门直播       | `.academy-section` + `.academy-live-*` | 主卡片+侧栏   |
| 知识专区       | `.academy-knowledge-grid`     | 2 列卡片       |
| 精品课程       | `.academy-course-grid`        | 3 列课程卡     |
| 行业报告       | `.academy-report-grid`        | 3 列报告卡     |

以上为培训学院页面（ref 目录下 `#/academy`）的完整布局说明。
